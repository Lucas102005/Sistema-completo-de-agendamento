# Sistema Completo de Agendamento + Loja (MSAS)

Sistema web completo para pequenos negócios (barbearias, salões, clínicas, etc).

## 🎯 Principais Funcionalidades

- Agendamento online com verificação de horários disponíveis
- Painel administrativo completo
- Controle de estoque com baixa automática
- Loja virtual integrada (venda de produtos)
- Notificações via WhatsApp
- Relatórios financeiros simples
- Upload de imagens de produtos

## 🛠 Tecnologias Utilizadas

- **Backend**: Node.js + Express
- **Banco de dados**: SQLite
- **Frontend**: HTML, CSS e JavaScript puro
- **Autenticação**: JWT
- **Upload de arquivos**: Multer

## 🚀 Como rodar localmente

1. Clone o repositório:
```Bash
   git clone https://github.com/Lucas102005/Sistema-completo-de-agendamento.git
```
2. Entre na pasta backend:
```Bash
   cd Sistema-completo-de-agendamento/backend
   npm install
```
3. Inicie o servidor:
```Bash
  node server.js
```

4. Acesse no navegador:
* Agendamento: http://localhost:3000/agenda.html
* Loja: http://localhost:3000/loja.html
* Admin: http://localhost:3000/admin.html


5. Login Admin padrão:
* Usuário: admin
